**Basic checklists:**
- [ ] normal
- [/] / - incomplete
- [x] x - done
- [-] -  canceled
- [s] s - forwarded
- [<] < - scheduling

**Extras checklists:**
- [?] ? - question
- [!] ! - exclamation
- [*] * - star
- ["] " - quote
- [l] l - location
- [b] b - bookmark
- [i] i - information
- [S] S - savings
- [I] I - idea
- [p] p - thumbs up
- [c] c - thumbs down
