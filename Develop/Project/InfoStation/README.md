### **后端技术栈**

- Spring Boot 2.7 框架
- springboot-init 项目模板
- MySQL 数据库
- Elastic Stack
    - Elasticsearch 搜索引擎
    - Logstash 数据管道
    - Kibana 数据可视化
- 数据抓取
    - 离线和实时抓取
    - Jsoup 和 HttpClient 库
- 设计模式
    - 门面模式
    - 适配器模式
    - 注册器模式
- 数据同步（4 种方式）
    - 定时
    - 双写
    - Logstash
    - Canal
- JMeter 压力测试
